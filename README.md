### 安装说明(from suyan)

1. fork库到自己的github
2. 修改名字为：`username.github.io`
3. clone库到本地，参考`_posts`中的目录结构自己创建适合自己的文章目录结构，修改文章栏目请移步/_includes/nav.html
4. 若使用独立域名请配置CNAME文件
5. 修改`_config.yml`配置项
6. It's done!


## 增加highlight.js，可在`_config.yml`配置风格，默认为monokai_sublime。
